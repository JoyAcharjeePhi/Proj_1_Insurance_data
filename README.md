# Proj_1_Insurance_data
End-to-end insurance risk analysis using Databricks (PySpark), Power BI, and Copilot Studio.

## 🚀 Automated Workflow
- **CI/CD:** Notebooks are synced directly from **Databricks** to this repository via Git integration.
- **Version Control:** All data engineering logic is versioned, allowing for collaborative development.
- **BI Asset:** The `.pdf` file is included in the `/reports` directory for full transparency of the analytical layer.
